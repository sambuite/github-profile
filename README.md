## Installation

### Step by step to run the application on your machine:

Clone this repository:

```
  git clone https://github.com/sambuite/github-profile.git

  cd github-profile
```

Run the following scripts :

```
  // Install the dependencies
  npm install
  
  // Start the application
  npm start
```
